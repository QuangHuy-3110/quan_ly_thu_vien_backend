# Ứng dụng quản lý thư viện - Backend

Đây là phần backend API của ứng dụng quản lý thư viện, được xây dựng bằng Node.js / Express.

## Cài đặt và sử dụng

1. Cài đặt các thành phần phụ thuộc:
   ```bash
   npm install
   ```

2. Chạy server trong môi trường phát triển:
   ```bash
   npm run dev
   ```
   Hoặc chạy ứng dụng:
   ```bash
   npm start
   ```

## Chức năng
- Cung cấp API phục vụ quản lý thư viện (Quản lý sách, mượn trả, người dùng, v.v.)
- Kết nối CSDL.
